GitHub+Hexo 搭建个人博客

Github Pages就是github提供的一个静态服务器，可以通过github给你提供的的二级域名来访问你的文件。

启用github page

  1. 新建仓库。名称为你的用户名.github.io的仓库。（一定要是你的github名！）,记得勾选readme
  2. 就好了，新建一个index.html，访问pangjunpeng.github.io试试看吧

总之就是可以放一个静态的html上去，你就可以访问了

而hexo，就是一个可以让你写markdown，然后编译为静态html的工具

Hexo

一个主要关注内容，而不用让你花太多心思在界面设计上的利器。
